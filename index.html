A personal-branding system that interrogates a professional's thinking, refuses vague answers, and distils a clear, distinct brand — then turns it into a brief their podcast producer, PR firm, or writer can act on.

Source repository · The Brand Company · 2025

What's in this repository

01

The Engine

An interactive, AI-powered app that challenges each answer, pushes for a distinct position and a single word to own, and synthesises a brief. Runs as a Claude artifact.

02

The Brief Generator

A Python tool that turns the Engine's exported answers into a branded PDF and an editable Word document, in The Brand Company's Swiss style.

How it flows

  1. A professional completes the Engine. It challenges every clarity answer and hard-gates the positioning and the word they own.
  2. The Engine distils a brief — positioning, the word, audience, voice, bios, talking points, sound bites, angles.
  3. They click Export for brief, which downloads the answers as a JSON file.
  4. The Brief Generator reads that JSON and produces a TBC-branded PDF and an editable Word document.

Run the generator

cd generator
pip install -r requirements.txt

# uses the bundled sample, writes a PDF + Word file here
python3 build_tbc_brief.py

# or use a real export
python3 build_tbc_brief.py path/to/visible-trusted-brief-data.json "Real Name"

Sharing with collaborators

This repository is the source, not the way to hand the tool to non-technical people. For partners who just need to use the Engine, publish the artifact in Claude and send them the link — they click and use it, with nothing to install. Running the Engine on a public website instead would require a small backend proxy holding an API key. See docs/PROJECT_HANDOFF.md for the full reasoning.
